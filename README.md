1. ASSIGNMENT:
   Analysing a product sales dataset from Kaggle for patterns and trends about the different products and brands being sold.
2. SOFTWARE:
 MS EXCEL, MY SQL SEVER, POWER BI
3. Overview:
 The steps taken to complete this analysis project 
- 	Importing the data to MySQL SERVER to create the database, write queries and create appropriate tables.
- 		Import the SQL database to Power Query for data cleaning and transforming.
- 			Using Power BI to create data visualizations and build 2 dashboards.
- 				Reporting on insights from the data.
